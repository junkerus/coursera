Coercion

Tipo de coercion:

  1. implicita, ejemplo: var a = 10; var b = a + "10"; b es un string porque se hace la conversion implicita.
  2. explicita, ejemplo: var a = 10; var b = String(a); b es un string porque se hace la conversion explicita.
  3. no hay coercion

Operadores de comparacion

  • ==, igualdad, compara si dos valores son iguales
  • ===, igualdad estricta, compara si dos valores son iguales y tienen el mismo tipo
  • !=, desigualdad, compara si dos valores son diferentes
  • !==, desigualdad estricta, compara si dos valores son diferentes y tienen el mismo tipo
  • <, menor que, compara si un valor es menor que otro
  • <=, menor o igual que, compara si un valor es menor o igual que otro
  • > mayor que, compara si un valor es mayor que otro
  • >=, mayor o igual que, compara si un valor es mayor o igual que otro

Firts Class Objects Objeto de primera Clase

una función puede ser pasada como argumento a otras funciones, puede ser retornada por otra función y puede ser asignada a una variable.

ejemplo:

function diHola() {
return "Hola ";
}
function saludar(saludo, nombre) {

Pasamos `diHola` y `JavaScript!` como argumento de la función `saludar`.

Una función que pasamos como argumento a otra función, se llama Callback function, funcion de retorno. diHola es una función Callback.

Una función Una función que devuelve una función se llama Higher-Order Function, funcion de primer orden.

Metodos: Bind, call y apply

bind, crea una nueva función, que cuando es llamada, asigna a su operador this el valor entregado, con una secuencia de argumentos dados precediendo a cualquiera entregados cuando la función es llamada.

call, llama a una función con un valor dado this y con argumentos provistos individualmente

apply, invoca una determinada función asignando explícitamente el objeto this y un array o similar (array like object) como parámetros (argumentos) para dicha función.
